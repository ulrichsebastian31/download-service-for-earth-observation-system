Overview

This test suite is based on the following OGC specifications:

  • OpenGIS Web Services Common Specification [OGC 05-008c1]
  • OWS Common Implementation Specification [OGC 06-121r9]

The conformance test cover basic search capabilities.

What is tested

  • RESET Core Conformance Class
  • RESET Core Capabilities Tests
  • GET Capabilities
  • GET Capabilities ServiceIdentification
  • GET Capabilities ServiceProvider
  • GET Capabilities Contents
  • Capabilities non nominal conditions
  • RESET Core Procedures Tests
  • DSEO Core Conformance Class
  • DSEO Core Capabilities Tests
  • GET Capabilities
  • GET Capabilities non nominal conditions
  • DSEO Core Product Download Tests
  • GET Single File Product
  • GET File Product non nominal conditions
  • GET Multi File Product
